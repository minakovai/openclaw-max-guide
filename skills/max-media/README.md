# MAX Media Skill

Скилл для работы с медиа-файлами в MAX мессенджере.

## Возможности

- ✅ Получение изображений от пользователей
- ✅ Отправка изображений пользователям
- ✅ Поддержка форматов: JPEG, PNG, GIF

## Как это работает

### Получение изображений (автоматически)

Когда пользователь отправляет картинку в чат MAX:

1. Плагин `@olegbalbekov/openclaw-max` автоматически скачивает изображение
2. OpenClaw передаёт путь к файлу в контекст разговора
3. AI-модель может проанализировать изображение (если поддерживает vision)
4. Ответ отправляется пользователю

### Отправка изображений

В ответе OpenClaw используйте формат:

```
MEDIA:https://example.com/image.jpg
Ваш текстовый ответ
```

Или для локальных файлов:

```
MEDIA:/path/to/local/image.png
Ваш текстовый ответ
```

## Пример использования в коде

```javascript
// В ответе агента
return {
  content: [
    { type: "text", text: "Вот изображение:" },
    { type: "image", url: "https://example.com/image.jpg" }
  ]
};
```

## Конфигурация

В `openclaw.json` медиа уже настроены:

```json
{
  "tools": {
    "media": {
      "images": {
        "enabled": true,
        "maxBytes": 10485760,
        "supportedFormats": ["jpg", "jpeg", "png", "gif", "webp"]
      }
    }
  }
}
```

## Ограничения MAX API

- Максимальный размер файла: ~20 MB (зависит от настроек бота)
- Рекомендуемые форматы: JPEG, PNG
- GIF поддерживается, но может быть ограничен по размеру
